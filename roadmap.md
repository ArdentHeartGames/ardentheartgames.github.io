# Development Roadmap

Our high-level roadmap for the Ardent Heart Editor and Unicorn Game Engine. This roadmap is subject to change based on community feedback and development priorities.

## Current Status: Alpha Development

We are currently in active development, working towards our first alpha release. The editor is functional for basic 2D game development, and we're continuously adding features and improvements.

## Release Schedule

We follow an iterative development approach:

- **Weekly Updates**: Bug fixes and small improvements
- **Monthly Milestones**: New features and capabilities
- **Quarterly Releases**: Major version updates with significant features

### Upcoming Milestones

1. **Alpha 0.1** - Core 2D editor (Q1 2026)
2. **Alpha 0.2** - Enhanced 2D features and physics (Q2 2026)
3. **Beta 0.1** - ThreeJS integration and 3D support (Q3 2026)
4. **Beta 0.2** - Advanced tools and optimization (Q4 2026)
5. **Version 1.0** - Production-ready release (2027)

*Note: Dates are approximate and subject to change*
